                        JavaScript Map

    In JavaScript, a Map is a built-in object that allows you to store key-value pairs where 
    keys can be of any data type (primitive or non-primitive). 

    # Features of Map : 
    1. Preserves Insertion Order: The order of elements is guaranteed to be the same as their insertion order.
    2. Flexible Key Types: Keys can be of any data type (e.g., string, object, number, etc.).
    3. Size Property: Maps have a size property that gives the number of key-value pairs.
    4. Efficient: Provides better performance for frequent additions and removals of key-value pairs.

    # Creating a Map
    You can create a Map using the new Map() constructor.
    syntax : 
    let map = new Map();


# Map Methods:-
---------------
1. set(key, value)
Adds or updates a key-value pair in the Map.

2. get(key)
Retrieves the value associated with the specified key.

3. has(key)
Checks if a key exists in the Map.

4. delete(key)
Removes a key-value pair from the Map.

5. clear()
Removes all key-value pairs from the Map.

6. size
Returns the number of key-value pairs in the Map.

7. Iteration Methods
keys(): Returns an iterator for the keys.
values(): Returns an iterator for the values.
entries(): Returns an iterator for [key, value] pairs.
forEach(callback): Executes a callback function for each key-value pair.
